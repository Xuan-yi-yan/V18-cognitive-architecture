# P2 词语→字符 反向解析 点级卡片 v1.0
# 日期: 2026-06-03

## 基本定义
- 点名称: WordToCharDecoder
- 所属位置: 基础处理栈 / 字符-词语族 / 第2层(闭环验证层)
- 功能: 将P1输出的32D词语向量, 反向拆解为两个原始字符的32D向量
- 角色: P1的闭环验证点, 证明P1的词向量包含了可解包的字符信息

## 接口协议

### 输入
P1输出的32D词语向量(冻结P1权重产生)

### 输出
字1特征(8D位置+24D语义=32D) + 字2特征(8D位置+24D语义=32D)

### 训练约束
- P1权重完全冻结(requires_grad=False)
- 仅P2的参数可训练
- P2必须纯粹从32D词向量中还原字符信息

## 架构
32D词向量 → Shared(Linear→128→ReLU→128→ReLU) → split:
  ├─ Char1 Head(Linear 128→32)
  └─ Char2 Head(Linear 128→32)

## 训练
- Loss: 1 - (cos_sim(pred_c1,real_c1) + cos_sim(pred_c2,real_c2)) / 2
- 评估: 平均余弦相似度
- 验收: 平均余弦相似度 > 90%
