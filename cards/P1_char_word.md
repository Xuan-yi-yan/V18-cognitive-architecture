# P1 字符→词语 点级卡片 v1.1
# 日期: 2026-06-03

## 基本定义
- 点名称: CharToWordPoint
- 所属位置: 基础处理栈 / 字符-词语族 / 第1层(入口层)
- 功能: 两个字符输入 → 交叉注意力融合 → 32D词语预测向量
- 角色: V18管线入口点, 无上游

## 接口协议

### 输入
两个字符, 每个: 8D位置(sin/cos) + 24D语义(可学习Embedding) = 32D
数据格式: !词@字1@字2 (每行一条)

### 输出
32D词语预测向量: 8D位置 + 24D语义
- 位置: pos1+pos2 → Linear(16→8) → L2归一化
- 语义: content均值 + 注意力content残差 + 元学习调制 → L2归一化

### 下游
输出32D词向量 → 词语-字符族(反向验证层)

## 架构规格

### 交叉注意力
- 头数: 16
- 内部维度: 64D (神经网络)
- 每头: 4D
- Q = 初始字符数据(32D → Linear → 64D)
- K = 初始字符数据(32D → Linear → 64D)
- V = 初始字符数据(32D → Linear → 64D)
- 输出: 64D → Linear → 32D + 残差

### 调制链
探索区(loss → 8D) → 元学习区(8D → 24D Tanh) → 内容融合调制

### 训练
- 目标: word_embeddings[word_id] (独立可训练词嵌入)
- Loss: 1 - Pearson_r(pred, target)
- 评估: 余弦相似度检索, Top-1 > 80%
- 优化器: Adam lr=0.001

## 日志要求
- GPU显存(allocated/reserved/peak)
- 计算时间(ms)
- 输入字符+目标词
- 预测↔目标余弦相似度
- 元学习调制因子(24D范围+TOP5)
- 探索区调制因子(8D全部值)
- 交叉注意力矩阵(16头均值, 2×2)

## 验收
- Top-1词检索 > 80%
- Top-3 > 95%
- 失败类型: 随机乱词 / 顺序错误 / 语义混淆
